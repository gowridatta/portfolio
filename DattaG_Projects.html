Scratch Maze (Computer Science/Software Engineering)

August - September, 2017

The objective of this project was to use the web application Scratch to create a fully functioning maze. My role in this project was creating the maze and the functionalities by using the code blocks. The maze had to function such that if the person touches the walls of the maze, the level restarts. A secret level was added as well, adding a new touch to the maze. Challenges we had included the creature traveling the maze being too big to fit, and the background being black (the color of the walls, which causes the level to restart. The maze's code and structure was revised until the entire project was free of glitches. Through this project, I was able to learn more about the basics of programming through coding blocks, including variable roles and if-statements.
